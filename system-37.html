Link 1: Eyeballs
Cols C, D, E, F
01
Pull Instagram Reel views + followers
Open IG app. Tap profile > Professional dashboard > Insights. Tap "Content you shared" > filter "Reels" > set "Last 7 days." Sum the Plays/Views for all Reels posted this week. Then read follower count from your profile header.
IG App > Profile > Insights > Content > Reels > Last 7 days Col C: IG Reel Views Col D: IG Followers
Read: Total "Plays" across all Reels this week (sum them). Read: "Followers" number from profile. Enter both.
02
Pull YouTube Shorts views
Open studio.youtube.com. Click "Analytics" in left sidebar. Set date range to "Last 7 days" (top right). Read the "Views" number on the Overview card.
studio.youtube.com > Analytics > Last 7 days Col E: YT Views
Read: "Views" number on the Overview card. This is total channel views for the week. Enter this number.
03
Pull publication visitors + top post
Log into Beehiiv. Click "Website" > "Analytics." Set to "Last 7 days." Read "Unique Visitors" at the top. Then scroll below the chart to the "Top Pages" table. Row 1 = your top post. Write down the post title (skip "/" or "/subscribe" if they appear first).
Beehiiv > Website > Analytics > Last 7 days Col F: Pub Visitors Col Q: Top Post
Read: "Unique Visitors" number at top = Col F. Read: Title of #1 post in "Top Pages" table = Col Q.

Link 2: Actions
Cols G, H
04
Pull IG engagement rate
In IG Insights (same app from Step 1), tap "Accounts engaged" or look for Engagement rate. Set to "Last 7 days." If not shown directly: (Likes + Comments + Saves + Shares) / Reach = Engagement %.
IG App > Insights > Accounts engaged > Last 7 days Col G: Engagement %
Read: Engagement rate as decimal. 3.5% = enter 0.035. Cell auto-formats to percentage.
05
Pull ManyChat keyword triggers
Log into ManyChat. Click "Analytics" in left sidebar (or Automation > Flows). Check each keyword flow's trigger count for the last 7 days. Add up ALL keyword triggers: AUTOPILOT + TICKING + SCORE + MYNUMBER + COPY + SEED + PAYDAY + any others.
manychat.com > Analytics > Last 7 days (or Automation > Flows > each keyword) Col H: ManyChat Triggers
Read: Total keyword triggers across all flows. If AUTOPILOT=5, SCORE=3, COPY=2, enter 10.

Link 3: Tool Use (North Star Zone)
Cols I, J
06
Pull tool visits from Supabase
Log into Supabase. Select your KTM project. Click "SQL Editor" in the left sidebar. Paste this query and click "Run":
supabase.com/dashboard > SQL Editor
SELECT COUNT(*) AS tool_visits_7d FROM visits WHERE created_at >= NOW() - INTERVAL '7 days';
Col I: Tool Visits
Read: The single number returned. This is total page loads across all 7 CashVault tools. Enter it.
07
★ Pull tool completions (NORTH STAR)
Same Supabase SQL Editor. Paste this query and click "Run":
supabase.com/dashboard > SQL Editor
SELECT COUNT(*) AS completions_7d FROM tool_completions WHERE created_at >= NOW() - INTERVAL '7 days';
Col J: Tool Completions ★
Read: The single number returned. This is THE number. How many people finished a CashVault tool this week. Enter it.
This is the North Star. Every other metric in this protocol exists to feed this number. If this number isn't growing, nothing else matters until you fix it.

Revenue Velocity Chain Rules
RULE 1: Double Down
IF Col Q (Top Post) matches a specific pillar AND Step 10 click map shows the matching tool got the most clicks: next week, feature that same pillar with that tool's keyword CTA. Your audience is voting. Listen.
RULE 2: Subject Line Intervention
IF Col N (Open Rate %) drops 3 consecutive weeks: subject lines are going stale. Test a different format. Number-based ("$840/year in subscriptions") vs question ("When did you last check your credit?") vs statement ("Your paycheck has 30 holes").
Reference: System 14 A/B testing structure
RULE 3: Conversion Leak
IF Col F (Pub Visitors) trending up BUT Col K (New Subs) flat or declining: traffic is coming but not converting. Audit the email capture popup on publication site. Is the Beehiiv subscribe form visible and firing?
Reference: System 3 QA protocol
RULE 4: CTA Mismatch
IF Col O (Click Rate %) drops below 2% for 2 consecutive weeks: CTA is buried or irrelevant. Next email: move the tool link to paragraph 2, not the bottom. Match the tool to the email's pillar.
RULE 5: CTA Invisibility
IF Col C (IG Reel Views) growing BUT Col H (ManyChat Triggers) flat: your content CTA is invisible. People watch but don't act. Test: move CTA keyword to the first 3 seconds as on-screen text overlay. Spoken CTA alone isn't enough.
RULE 6: Tool UX Drop-off
IF Col I (Tool Visits) healthy BUT Col J (Tool Completions) low (completion rate under 15%): tool UX is losing people at a specific step. Run the drop-off SQL query in Supabase to find the exact step. Fix that step before sending more traffic.
SQL: SELECT step, COUNT(*) FROM funnel_events WHERE created_at >= NOW() - INTERVAL '7 days' GROUP BY step ORDER BY step;
RULE 7: Email Gate Failure
IF Col J (Tool Completions) growing BUT Col K (New Subs) flat: email gate popup is broken or poorly timed. Audit: is popup firing after the result screen? Is the value proposition clear? Is the Beehiiv integration receiving the data?
Reference: System 3 QA protocol for CashVault tools
