Weekly Protocol
0%
01
Pull subscriber numbers
Write down New Subscribers (last 7 days) and Total Subscribers.
Beehiiv > Audience > Overview Col C: New Subs Col D: Total List
02
Pull this week's email performance
Open this week's sent email. Write down Open Rate % and Click Rate %.
Beehiiv > Posts > [This Week's Email] Col E: Open Rate % Col F: Click Rate %
Column F is your email CTR. No separate CTR column needed. One email per week = one click rate number.
03
Check which tool got clicks
Still on the same email stats page from Step 2. Scroll down past the open/click summary. You will see a Click Map section showing every link in the email ranked by number of clicks. Find the kou.money link (e.g., kou.money/freedom-calculator, kou.money/budget-calculator). Note which tool URL got the most clicks. Do not log this number anywhere. Just remember it.
Same email stats page > scroll below open/click chart > Click Map section
This observation feeds the Decisions tab. If your email linked to multiple tools, the top-clicked one tells you which pillar your audience is voting for. If you only linked one tool, this step takes 5 seconds: confirm it got clicks, move on.
04
Pull publication unique visitors
Set date range to Last 7 days. Write down Unique Visitors.
Beehiiv > Website > Analytics Col N: Pub Visitors (NEW)
05
Identify top post by traffic
Still on the Website Analytics page from Step 4. Below the visitor chart, you will see a Top Pages table listing your publication URLs ranked by views. The #1 row is your highest-traffic post this week. Write down its title (not the URL slug, the actual post title). If the top result is your homepage or a generic page (like "/subscribe"), skip it and take the first actual post.
Beehiiv > Website > Analytics > scroll below visitor chart > Top Pages table > Row 1 Col O: Top Post (NEW)
Ignore this week: geographic data, device breakdown, browser stats, referrer sources. Those are monthly only (Monthly tab).
06
Enter everything into System 9
Open your Google Sheet. Find this week's row. Type in: C (New Subs), D (Total List), E (Open Rate), F (Click Rate), N (Pub Visitors), O (Top Post). Done.
Google Sheets > System 9 Dashboard
Columns G-M come from other sources (IG Insights, ManyChat, Supabase, manual). Fill those from their respective platforms during the same Sunday session.

Trend Formula for Column N

Week-over-week change. Put this in a helper column or next to N. Start from Row 6 (Row 5 has no prior week). Copy down.

=IF(N5=0,"—",(N6-N5)/N5)

Format as percentage. This tells you if publication traffic is growing, flat, or declining week over week.

Conditional Formatting for Column N
Apply to N6:N200 (same rules as your existing C-L formatting)
Green Growth above +2%  (#10B981)
Red Decline below -2%  (#EF4444)
Yellow Flat, within +/-2%  (#F59E0B)

Google Sheets: Format > Conditional Formatting > Custom formula.
Green: =AND(N6<>"",N5<>"",N5>0,(N6-N5)/N5>0.02)
Red: =AND(N6<>"",N5<>"",N5>0,(N6-N5)/N5<-0.02)
Yellow: =AND(N6<>"",N5<>"",N5>0,ABS((N6-N5)/N5)<=0.02)

Column O is text (post title). No formula or formatting needed. It exists so you can spot patterns over time: "Budget post was #1 three weeks in a row."

Content Decision Framework

Check after filling the dashboard. If none trigger, change nothing. That is also a valid decision.

IF / THEN Rules
RULE 1: Double Down
IF Column O (Top Post) is about a specific pillar AND Step 3 click map shows the matching tool got the most clicks: next week, feature that same pillar with that tool's keyword CTA. Your audience is voting. Listen.
RULE 2: Subject Line Intervention
IF Column E (Open Rate %) drops 3 consecutive weeks: your subject lines are going stale. Test a completely different format. Number-based ("$840/year in subscriptions") vs question ("When did you last check your credit?") vs statement ("Your paycheck has 30 holes").
Reference: System 14 A/B testing structure
RULE 3: Conversion Leak
IF Column N (Pub Visitors) trending up BUT Column C (New Subs) flat or declining: traffic is coming but not converting. Audit email capture popup on publication site. Is the Beehiiv subscribe form visible? Is it firing?
Reference: System 3 QA protocol
RULE 4: CTA Mismatch
IF Column F (Click Rate %) drops below 2% for 2 consecutive weeks: your CTA is buried or irrelevant to the email topic. Next email: move the tool link to paragraph 2, not the bottom. Match the tool to the email's pillar.
